# 参考资料目录

这里存放项目开发和部署过程中需要对齐的外部资料、接口说明和运维记录。

## 建议内容

- `dashscope-qwen-vl.md`：阿里 DashScope / Qwen VL 接口调用说明。
- `server-deploy.md`：公司华润雪花服务器部署记录。
- `nginx.md`：nginx 反向代理和上传大小配置。
- `systemd.md`：后端服务 systemd 配置说明。
- `test-images/`：用于人工验证的样例图片，不放敏感素材。

## 安全要求

- 不保存真实 API Key。
- 不保存服务器密码。
- 不保存包含敏感业务信息的原始图片，除非用户明确要求并确认存放位置。
- 示例配置使用占位符。
